## Technical Notes

### URL Patterns
- Clean URLs: `/ai` not `/ai.html`
- Lowercase, hyphenated: `/case-race` not `/CaseRace`
- Consistent structure

### Responsive Design
- Mobile-first approach
- Breakpoints: Mobile (320-767px), Tablet (768-1023px), Desktop (1024px+)
- All content accessible on all devices

### Performance
- Optimized images
- Lazy loading for below-fold content
- Fast page loads (space background as optimized SVG or compressed image)
